# HolidayPicker

## 프로젝트 소개
**HolidayPicker**는 사용자가 원하는 날짜를 설정하여 국경일(공휴일) 정보를 오름차순으로 조회할 수 있는 서비스입니다.  
TailwindCSS와 공공 API 사용에 익숙해지기 위한 실습 프로젝트로 개발되었습니다.

## 주요 기능
- 날짜 설정 기능
- 공휴일 오름차순 조회 기능
- 간편한 UI로 손쉬운 공휴일 검색

## 프로젝트 목적
- TailwindCSS를 활용한 스타일링 연습
- 외부 API 사용 및 데이터 처리 연습

## 기술 스택
- **Frontend**  
  - React
  - TailwindCSS
